# 🌱 Opti Crop – Smart Agriculture Production Optimisation Engine

## 📖 Overview

**Opti Crop** is a smart agriculture platform that helps farmers, researchers, and policymakers make data-driven decisions. Using agricultural and environmental data, the system provides intelligent recommendations to improve crop planning, productivity, and sustainable farming practices.

---

## ✨ Features

* 🌾 **Crop Recommendation** – Suggests the most suitable crops based on input conditions.
* 📈 **Yield Prediction** – Estimates expected crop yield using historical and environmental data.
* 🌱 **Crop Suitability Analysis** – Evaluates the suitability of land and climate for different crops.
* 📊 **Research & Policy Support** – Provides insights to assist researchers and policymakers in agricultural planning.

## 🚀 Getting Started

1. Clone the repository

   ```bash
   git clone https://github.com/Adityaduggani/OptiCrop-Smart-Agriculture-Production-Optimiser.git
   ```

2. Navigate to the project directory

   ```bash
   cd OptiCrop-Smart-Agriculture-Production-Optimiser
   ```

3. Install dependencies

   ```bash
   npm install
   ```

4. Configure environment variables in `.env.local`.

5. Start the development server

   ```bash
   npm run dev
   ```
